Scenario planning

An alternative way of dealing with uncertainty

Introduction

Saddam Hussein Invades Kuwait

  • Intelligence agencies sent Bush list of predictions.
  • None had first choice that Saddam would attack.
  • Why?
    • Groupthink = group thinking becomes homogeneous.
    • Minority opinions overrun by majority.

Scenario Planning: A Better Way?

  • Build possible scenarios and plan actions.
  • Advantages:
    • Allows minority opinions to have 'airtime'.
    • Predicting future hard; instead, scenarios bound the range of possible outcomes.
    • Participants explore problem space, gaining key insights.

Method 1

The Extreme Worlds Method

Overview of Method

  • Develop two extreme worlds (scenarios)
    • Should bound all possible scenarios.
  • Eigh distinct steps
    • Can retrace steps during process.
  • Multinational NZ Dairy Monopoly
  • Revenue of $20 billion
  • 30% of world's dairy
  • Recent Botulism scare

1/8 Concern & Horizon

  • Concern: The survival and profitability of Fonterra.

2/8 Identify Trends

  • Increasing dairy consumption (2% per year).
  • Preference to buy NZ dairy products.
  • Growing Asian market.

3/8 Identify Uncertainties

  • Consistent quality of Fonterr's products.
  • Strength of NZ dollar.
  • Ability to hold NZ monopoly.
  • Cow "fart tax" in NZ.

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the background.

Custom Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.addEventListener( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );

Slide Backgrounds

Set data-background="#007777" on a slide to change the full page background to the given color. All CSS color formats are supported.

Image Backgrounds

<section data-background="image.png">

Repeated Image Backgrounds

<section data-background="image.png" data-background-repeat="repeat" data-background-size="100px">

Background Transitions

Pass reveal.js the backgroundTransition: 'slide' config argument to make backgrounds slide rather than fade.

Background Transition Override

You can override background transitions per slide by using data-background-transition="slide".
